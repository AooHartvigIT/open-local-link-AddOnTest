
Open local link by clicking

This extension make it possible to open a local file or folder by clicking its link  (tag <a href="file://...">) in Web page.
When you click the lin, the local file would be opened by the application associated with the file if the file suffix is xlsx, docx or pdf, otherwise opened in Windows Explorer.

This extension requires installing native client application in the local PC.
The installation instruction will be displayed at the installation of this extension.

- the local link must be a <a> tag
- support multi-frame web page (frameset)
